| Content-Type             | 解析方式              | 返回类型    |
|:------------------------|:---------------------|:-----------|
| `application/json`      | `response.json()`    | 对象/数组   |
| `multipart/form-data`   | `response.formData()`| `FormData` |
| `text/plain`            | `response.text()`    | 字符串      |
| 其他                     | -                    | `undefined`|
