## Send Your First Request

Sending HTTP requests with Keq is simple — just call the corresponding method and `await` the result:

```typescript
import { request } from "keq"

const cats = await request.get("/cats")

console.log(`My lovely cats are: ${cats.map(cat => cat.name).join(", ")}`)
```

:::tip Automatic Response Parsing
Keq automatically parses the response body based on the `Content-Type` response header ([details](./responses/auto-parsing)), saving you from manually calling `.json()`.
:::
