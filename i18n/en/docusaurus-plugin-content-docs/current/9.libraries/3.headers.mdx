# @keq-request/headers

Provides middleware for modifying HTTP request headers.

## Installation

import InstallPackagesCode from '@site/docs/components/_install-packages-code.mdx'

<InstallPackagesCode packages="@keq-request/headers" />

## Usage

```typescript
import { request } from 'keq'
import {
  setHeader,
  setHeaders,
  appendHeader,
  appendHeaders,
  insertHeader,
  insertHeaders,
} from 'keq-headers'


request
  .use(setHeader('x-user-name', 'Kerry'))
  .use(appendHeader('x-user-name', 'Marry'))
  .use(insertHeader('x-user-name', 'J'))
  .use(setHeaders({
    'x-user-name': 'Kerry',
    'x-user-id': '1'
  }))
  .use(appendHeaders({
    'x-user-name': 'Marray',
    'x-user-id': '1',
  }))
  .use(insertHeaders({
    'x-user-name': 'J',
    'x-user-id': 1,
  }))
```

| **Middleware** | **Description**                                  |
| :------------- | :---------------------------------------- |
| setHeader      | Adds a header, overwriting if it already exists     |
| setHeaders     | Adds multiple headers, overwriting if they already exist |
| appendHeader   | Adds a header without affecting existing headers        |
| appendHeaders  | Adds multiple headers without affecting existing headers    |
| insertHeader   | Adds a header only if it doesn't already exist           |
| insertHeaders  | Adds multiple headers only if they don't already exist       |
