# Timeout Control

**`.timeout(milliseconds)`**

| **Parameter** | **Default** | **Description** |
| :------------ | :---------- | :-------------- |
| milliseconds  | `Infinity`  | Timeout period  |

## Example

<!-- prettier-ignore -->
```typescript
import { request } from 'keq'


try {
  await request
    .get('/cat')
    .timeout(1000)
} catch (err) {
  console.log(err)
}
```
